============= СПРАВКА ==============================================

------------- Как запускать программу
1. В модуле server запустить ChatServer.main
   Убедиться, что сервер поднят (см. коноль).
2. В модуле client запустить ChatClient.main
   Появится окошко клиента.
3. Общаться

------------- Как запустить несколько экземпляров клиента
1. Открыть класс ClientChat
2. В строке, содержащей сигнатуру метода main, слева есть зелёный треугольничек. Тыцнуть.
3. Modify run configurations -> Modify options
5. Поставить галку Allow multiple instances

------------- Как прописать зависимости между модулями
Конкетно в данном проекте необходимо из модулей ClientChat и ServerChat прописать
зависимоть от модуля network. Иначе из ClientChat и ServerChat будет невозможно увидеть
классы, интерфейсы, методы и константы network.
1. Правой кнопкой на модуль, откуда устанавливаем зависимости -> Open module settings.
2. Dependencies
3. Плюсик
4. Выбираем модуль, от которого будет зависеть текущий модуль. Например:
- clientChat должен зависеть от network;
- serverChat должен зависеть от network.
5. ОК.
Вуаля!
